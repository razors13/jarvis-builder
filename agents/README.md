# Directorio agents/

Contiene agentes de IA y componentes de automatización.

## Estructura

```
agents/
├── builder-agent.md      # Agente constructor (implementa tareas)
├── planner-agent.md      # Agente planificador (diseña estrategias)
├── repo-agent.md         # Agente de repositorio (revisa estructura)
└── README.md            # Documentación del directorio
```

## Agentes Implementados

- **builder-agent.md**: Ejecuta tareas de implementación
- **planner-agent.md**: Genera planes técnicos detallados
- **repo-agent.md**: Revisa y mejora estructura del proyecto

## Flujo de Trabajo

1. Planner Agent diseña el plan
2. Builder Agent implementa las tareas
3. Repo Agent revisa y optimiza la estructura
