# Directorio docs/

Contiene documentación de la API y especificaciones OpenAPI.

## Estructura

```
docs/
└── swagger.json          # Especificación OpenAPI de la API
```

## Contenido Planificado

- **swagger.json**: Documentación completa de endpoints, modelos y ejemplos de uso
- **API Reference**: Guía detallada de cada endpoint
- **Examples**: Ejemplos de código y casos de uso

## Herramientas Utilizadas

- **Swagger/OpenAPI**: Especificación estándar para documentación de APIs REST
- **swagger-jsdoc**: Generación automática de documentación desde comentarios de código
