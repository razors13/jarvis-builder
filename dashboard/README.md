# Directorio dashboard/

Contiene la interfaz de usuario y panel de control de la aplicación.

## Estructura

```
dashboard/
└── README.md            # Documentación del directorio
```

## Contenido Planificado

- **Frontend**: Interfaz web para interactuar con la API
- **Panel de Control**: Dashboard para monitorear y administrar
- **Documentación**: Guías de uso y configuración

## Tecnologías Sugeridas

- React/Vue.js - Framework frontend
- Tailwind CSS - Estilos
- Chart.js - Visualización de datos
- WebSocket - Actualizaciones en tiempo real
